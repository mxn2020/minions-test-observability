---
title: Installation
description: Install Minions Test-observability packages
---

## TypeScript / Node.js

```bash
npm install @minions-test-observability/sdk minions-sdk
```

## Python

```bash
pip install minions-test-observability
```

## CLI

```bash
npm install -g @minions-test-observability/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
