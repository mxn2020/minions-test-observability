# Minions Test-observability

Trace capture, tool call logs, reasoning timings, and latency breakdowns

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-test-observability/sdk`) — Core library for Node.js
- **Python SDK** (`minions-test-observability`) — Python bindings
- **CLI Tool** (`@minions-test-observability/cli`) — Command-line interface
- **Web App** — Interactive playground at [test-observability.minions.wtf](https://test-observability.minions.wtf)
